# Maintenance, Repair & Operations (MRO)

Comprehensive MRO management for maintenance consumables, spare parts, and repair operations, managed by the Manufacturing Service (EAM module) with Commerce Service integration.

| Aspect | Implementation |
|--------|---------------|
| MRO Catalog | Centralized catalog of maintenance supplies, spare parts, and consumables with classification and search |
| MRO Procurement | Automated reorder point-based procurement with blanket purchase agreements and vendor-managed inventory |
| Spare Parts Inventory | Dedicated spare parts inventory management with criticality classification, min/max replenishment, and obsolescence tracking |
| Tool Management | Maintenance tool tracking with calibration schedules, certification management, and checkout/check-in |
| Repair Order Management | Repair work orders with failure codes, repair estimates, actual cost tracking, and turnaround time monitoring |
| Overhaul Management | Complex overhaul project management with task breakdown, parts kitting, and progress tracking |
| Rotable Management | Rotable (exchangeable) component tracking with repair cycle management, exchange pools, and core tracking |
| Vendor Repair Tracking | External vendor repair tracking with RMA management, turnaround time SLAs, and quality verification |
| MRO Analytics | MRO spend analysis, stockout frequency, repair cost trends, vendor performance, and asset availability dashboards |
| Integration | Seamless integration with EAM for planned maintenance, Finance for cost accounting, and Commerce for procurement |

*See [Warranty Management](warranty.md) for warranty claims on repairs. See [Enterprise Asset Management](../services/manufacturing.md) for asset lifecycle. See [Architecture Overview](../architecture/overview.md) for system context.*
