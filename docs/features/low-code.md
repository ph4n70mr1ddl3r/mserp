# Low-Code Application Builder

A visual application builder enabling business users to create custom applications without coding, managed by the Platform Service.

| Aspect | Implementation |
|--------|---------------|
| Page Designer | Drag-and-drop UI builder with pre-built components (forms, tables, charts, calendars, Kanban boards) |
| Business Objects | Custom entities with fields, validations, relationships, and computed fields stored in Platform DB |
| Workflow Integration | Custom workflows via BPMN designer connected to business objects with configurable triggers |
| Data Binding | Connect custom UI to any MSERP API endpoint with declarative data source configuration |
| Form Builder | Visual form designer with field types, validation rules, conditional visibility, and multi-step forms |
| Business Logic | Visual rule builder for conditional logic, calculations, and data transformations without code |
| Dashboard Builder | Custom dashboard builder with configurable widgets, charts, KPI cards, and real-time data feeds |
| Mobile Support | Responsive design with mobile-optimized layouts and touch-friendly components |
| Integration | REST API connector for external data sources with authentication, mapping, and polling/webhook triggers |
| Role-Based Access | Application-level access control using existing RBAC/ABAC roles and permissions |
| Publishing | Apps published per-tenant with version control, rollback, and staged rollout |
| Sandbox | Test customizations in sandbox before publishing to production with data masking |
| Theme Engine | Configurable themes with corporate branding, colors, logos, and custom CSS injection |
| Extension Points | JavaScript extension points for advanced customization when visual builder reaches limits |
| Template Marketplace | Pre-built application templates for common use cases (CRM, project tracking, inventory) |
| Analytics | Usage analytics for custom applications with adoption tracking and performance monitoring |

*See [Workflow & BPM Engine](../services/workflow.md) for BPMN workflows. See [API Marketplace](api-marketplace.md) for API catalog. See [Architecture Overview](../architecture/overview.md) for system context.*
